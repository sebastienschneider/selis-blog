# About

Hi, I'm Seli.

I write about technology, software development, and whatever else I find interesting. This blog is a place to share ideas, document what I learn, and think out loud.

I believe in keeping things simple — both in code and on the web. That's why this site has no JavaScript, no tracking, and no ads. Just clean, readable content.

## About This Site

This blog is powered by [Hugo](https://gohugo.io), a fast static site generator written in Go. The theme is [Bear Blog](https://github.com/janraasch/hugo-bearblog/), which prioritizes speed and readability above all else.

The site is deployed on [Netlify](https://www.netlify.com) with continuous deployment from GitHub. Every push to the main branch triggers a new build and deploy automatically.
